## Introduction[<span style="margin-left: 10px; text-decoration: none;">📌</span>](#introduction)

Vous en avez marre de devoir apprendre une grosse quantité de cours et vous ne trouvez pas de solution pour résumer votre cours automatiquement ? Vous êtes au bon endroit !

Ce projet a pour but de résumer des cours, sous format PDF, en utilisant un modèle machine learning (LLM), [Mistral](https://mistral.ai).

À la différence des solutions existantes, le processus est optimisé afin de résumer des cours de manière efficace et rapide.

## Pourquoi ce projet[<span style="margin-left: 10px; text-decoration: none;">🤔</span>](#pourquoi-ce-projet)

Ce projet est venu d'une demande d'un étudiant en médecine qui prenait beaucoup de retard et qui souhaitait résumer ces cours rapidement. Il a d'abord essayé de le résumer avec ChatGPT, sauf qu'il n'était pas précis, du fait que les cours font beaucoup trop de pages.

C'est ainsi que j'ai décidé de faire une solution optimisé, afin de garantir une précision optimale, peu importe le nombre de pages du cours.
